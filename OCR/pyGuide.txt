edittrans.py
Main GUI for editing pages.

fileutil.py
Methods for translating between file naming conventions, and getters for absolute file paths. Needs converted to relative file paths.

getcodec.py
Get all characters occuring in a transcription and store them in a codec file.

lev.py
Calculate edit distance between two one line files. Calls function in stringutil.

postproc.py
Connect transcription to image box.

sellines.py
GUI interface for selecting which lines to transcribe and which to discard.

septrans.py
Split a transcription into files for each line according to the naming convention.

stringutil.py
As it says. Lots of short utility functions.

subtrans.py
Replace one character in a page transcription with another. Needs a closer look.
